- Net: per CONSTITUTION III.1 / VI.1 the **InvisibleHand tool is a tombstone** — the prior matrix's "diminished, Skimlinks-owned, minimal updates" diagnosis resolved into a full rebrand. Do not present "InvisibleHand" as a live tool.
